\section{Обзор существующих методов}
\begin{frame}{Обзор существующих методов}
    \begin{itemize}
        \item Многие объективные оценки качества являются дифференцируемыми по входным параметрам (например, LPIPS, SSIM).
        \item Состязательная атака путем использования градиента оценки качества была предложена в работе \alert{“Maximum Differentiation (MAD)”}.
        \item Метод “VMAF with video color” использует набор стандартных преобразований для изображений (повышение резкости, выравнивание гистограмм).
    \end{itemize}
\end{frame}

\section{Предложенный метод}
\begin{frame}{Предложенный метод}
    \begin{itemize}
        \item Метод основан на обучении промежуточной аппроксимации рассматриваемой оценки качества VMAF.
        \item Задача — максимизация оценки качества по методу при сохранении качества кодирования.
        \item Используется архитектура на базе \alert{U-Net} для предобрабатывающей нейронной сети.
    \end{itemize}
\end{frame}

\section{Модификация обучающей выборки}
\begin{frame}{Модификация обучающей выборки}
    \begin{itemize}
        \item Повышение значений VMAF на тестовой выборке после применения предложенного метода.
        \item \alert{Некоторый процент примеров оказался сложным для модели}, и предобработка не увеличивает, а уменьшает значение оценки качества VMAF.
    \end{itemize}
\end{frame}

\section{Субъективная оценка}
\begin{frame}{Субъективная оценка}
    \begin{itemize}
        \item Модели на Pascal Voc создавали яркие цветные пятна и ложные цветовые границы.
        \item Другие типы искажений включали цветные артефакты на некоторых границах изображений.
        \item У сети, обученной на Vimeo 90K, отсутствуют ложные цветовые границы.
    \end{itemize}

\section{Результаты}
\begin{frame}{Результаты}
    \begin{itemize}
        \item Фильтрация обучающей выборки оказала наибольшее влияние на эффективность.
        \item Добавление SSIM-регуляризации позволило избавиться от цветовых пятен.
        \item Наилучшая из обученных моделей показывает BSQ-Rate равный \alert{0.541}.
    \end{itemize}
